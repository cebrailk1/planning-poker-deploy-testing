# Software to play planning Poker online with your team.

# Description

## 🃏 Planning Poker

Planning Poker (also known as Scrum Poker) is an Agile estimation technique used by teams to assess the effort or complexity of tasks.

### How It Works
1. A task or user story is presented to the team.
2. Each participant privately selects a card (usually from the Fibonacci sequence: 1, 2, 3, 5, 8, 13, …).
3. Everyone reveals their card at the same time.
4. If estimates differ, the team discusses reasoning and knowledge gaps.
5. The process repeats until a consensus is reached.

### Benefits
- **Collaborative:** Encourages discussion and shared understanding.
- **Unbiased:** Prevents anchoring by revealing all estimates simultaneously.
- **Accurate:** Leads to better, more democratic estimates over time.
